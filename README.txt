NewMod 1.0 для BombSquad

На данный момент нет подробных инструкций для пользования модом. По всем вопросам пишите в тг @BSFanta

Чтобы получить привилегию HOST на созданном сервере, зайдите и введите команду
/promo host1258
Этот промокод работает один раз и выдаёт роль HOST использовавшему ему игроку.
После этого роли можно изменить в файле playersData/roles.json
Напишите /reset; end чтобы сбросить топ и начать новый сезон
